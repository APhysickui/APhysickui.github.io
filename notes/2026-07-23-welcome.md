# Welcome

This is my **Notes** space — a place for essays, reading notes, and
half-formed thoughts on physics and code.

## How it works

I write in Markdown. Each file in the `notes/` folder becomes its own page,
styled to match the rest of the site. To add a note I just:

1. Drop a new `.md` file into `notes/`.
2. Run `python3 build_notes.py`.
3. Commit and push.

## What you can write

- **Bold** and *italic* text
- Lists, like this one
- `inline code` and code blocks
- [Links](https://github.com/APhysickui)

```python
def hall_response(material):
    return screen(material)
```

> Blockquotes work too — good for citations.
